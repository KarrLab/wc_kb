# wc_kb

Tools for building and describing knowledge bases for whole-cell models

## Installation

```
pip install git+https://github.com/KarrLab/wc_kb.git#egg=wc_kb
```

## Documentation
Please see the [API documentation](https://yinhoon-wc-kb.readthedocs.io).
